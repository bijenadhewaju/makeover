# Makeover Me 💄✨

**Makeover Me** is a frontend web application built with React and Tailwind CSS. The UI was developed from a provided Figma design and integrated with backend APIs to fetch and display dynamic data.

---

## 🚀 Live Demo

🔗 https://makeover-seven.vercel.app/

---

## 📌 Features

- Responsive and modern user interface
- Built from Figma design
- Component-based React structure
- Styled with Tailwind CSS
- Consumes provided APIs for dynamic content

---

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- Vite
